#Nume Proiect

Sistem informatic pentru optimizarea rutelor de curierat

#Descriere

Aplicatie inteligenta care organizeaza automat munca curierilor. Dispecerul introduce adresele, iar sistemul calculeaza instantaneu cel mai rapid si scurt traseu pe harta. Astfel, soferul foloseste aplicatia mobila pentru a naviga eficient de la un client la altul, economisind timp, combustibil si reducand intarzierile coletelor.
